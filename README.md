## Install

Copy and run this script:  

```
git clone http://github.com/erming/dotfiles && ./dotfiles/install.sh
```

## Shortcuts

Useful `git` aliases:

```
a = add --all
b = branch
c = commit
d = diff
l = log --oneline -5
p = push
s = status
```
